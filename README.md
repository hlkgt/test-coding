## BeliBang soal test coding

# Cara Clone File

1. clone dari github berikut

```
git clone https://github.com/hlkgt/test-coding.git
```

2. pindah ke folder yang sudah diclone

```
cd test-koding
```

3. install package yang dibutuhkan

```
npm install
```

4. jalankan server lokal

```
npm run dec
```
